# 小黑键盘猫 (Bongo Cat Mver)

一个可爱的桌面键盘猫猫覆盖层工具，当你敲键盘或点击鼠标时，屏幕上会显示一只小猫跟着敲击对应的按键。

## 功能

- **键盘模式** — 实时显示键盘按键操作
- **鼠标模式** — 显示鼠标点击（左键、右键、侧键）
- **手柄模式** — 支持游戏手柄按键显示
- **Live2D 模型** — 可选 Live2D 动态猫娘模型
- **自定义配置** — 按键映射、窗口位置、颜色等均可调整

## 文件说明

| 文件 | 说明 |
|------|------|
| `Bongo Cat Mver.exe` | 主程序（桌面猫猫覆盖层） |
| `BongoCatUI.exe` | 设置/教程界面 |
| `config.json` | 配置文件（按键映射、窗口位置等） |
| `Resources/` | 字体和 logo 资源 |
| `img/` | 三种模式的图片素材（standard/keyboard/gamepad） |

## 三种模式

1. **标准模式 (standard)** — 全键盘 + 鼠标，适合日常使用
2. **键盘模式 (keyboard)** — 精简键盘布局
3. **手柄模式 (gamepad)** — 游戏手柄按键显示

## 使用方法

1. 运行 `Bongo Cat Mver.exe`
2. 右下角托盘图标出现小猫，右键可进入设置/切换模式
3. 敲击键盘或点击鼠标，猫猫会同步动作

> 对软件有疑问请右键托盘图标进入 UI 查看教程。
